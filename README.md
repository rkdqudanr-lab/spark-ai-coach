# 🚀 SPARK - 창업 준비 실행 코치

예비창업패키지 준비를 위한 AI 기반 실행 코치

## ✨ 주요 기능

- 🎯 매주 구체적인 도전과제 제시
- 💬 AI 기반 맞춤형 대화
- 📊 진행상황 추적
- 🔐 안전한 API 키 관리

## 🌐 배포 사이트

https://spark-ai-coach.vercel.app

## 🛠️ 로컬 실행

```bash
# 설치
npm install

# 실행
npm start
```

http://localhost:3000 에서 확인

## 📝 사용 방법

1. Claude API 키 발급 (console.anthropic.com)
2. 앱에서 API 키 입력
3. 대화 시작!

## 💡 기술 스택

- React 18
- Claude API (Sonnet 4)
- Tailwind CSS
- Lucide Icons
